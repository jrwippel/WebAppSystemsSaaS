FAQ

Perguntas frequentes

Preciso de cartão de crédito para o trial?
Não. O trial de 14 dias é completamente gratuito e não exige cartão. Você só informa dados de pagamento se decidir continuar.
Posso usar para qualquer tipo de negócio?
Sim. Embora seja muito usado por escritórios de advocacia, o ClockTrack funciona para qualquer negócio que precise controlar horas e recursos — agências, consultorias, freelancers e mais.
Meus dados ficam seguros?
Sim. Cada empresa tem seu próprio espaço isolado (multi-tenant). Os dados são armazenados no Microsoft Azure com backup automático e conexão criptografada.
Posso cancelar a qualquer momento?
Sim, sem multa e sem burocracia. Entre em contato e cancelamos imediatamente.
O sistema tem app mobile?
O ClockTrack é responsivo e funciona muito bem no celular pelo navegador. Um app nativo está no roadmap.
